# Chandana Sirimalla — Professional Portfolio

A highly polished, editorial-style professional portfolio application engineered to showcase modern full-stack development expertise, responsive design precision, and aesthetic UI craftsmanship. 

This portfolio acts as a live hub of development works, interactive skill metrics, and a dynamic customization engine tailored for potential employers and collaborators.

---

## 🎨 Visual Identity & Dynamic Theme Engine

The application is built on an editorial, dark-mode-first slate design system using **Tailwind CSS v4** native CSS variables. It features a floating custom theme engine allowing viewers to dynamically alter the accent profile to match their preferences:

-   🟢 **Emerald Forest** (Eco-Green, high-contrast, representing growth and stability)
-   🟣 **Cyber Lavender** (Retro-futuristic violet, sharp and modern)
-   🔵 **Oceanic Cyan** (Serene aqua highlight, optimized for deep readability)
-   🟡 **Sunset Amber** (Warm amber gold, organic and inviting)
-   🔴 **Crimson Rose** (Bold cherry pink, energetic and prominent)

The theme customizer updates the `--accent`, `--accent-dark`, and `--accent-glow` CSS variables across the entire shadow tree dynamically, transforming the scrollbars, visual glowing lights, active button states, and accent highlights instantly.

## 🚀 Quick Start (Local Development)

To run this repository locally, follow these steps:

### 1. Install Dependencies
```bash
npm install
```

### 2. Launch the Development Server
```bash
npm run dev
```
The application will boot and become available at `http://localhost:3000`.

### 3. Production Build
To compile optimized static assets ready for fast static hosts (such as Netlify, Vercel, or GitHub Pages):
```bash
npm run build
```
Compiled production-ready assets will be located inside the `/dist` directory.
